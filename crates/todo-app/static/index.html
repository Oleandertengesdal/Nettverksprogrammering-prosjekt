Shared Todos

Real-time peer-to-peer todos powered by CRDT state sync.

This frontend is transport-driven: it renders peer state received from the backend and sends local operations upstream. There is no local CRDT simulation in this page.

How to demo in 3 steps

  1. Press Connect to attach this client to the sync layer.
  2. Create and toggle todos from this client.
  3. Observe peer cards update live when remote peers join, leave, and edit.
